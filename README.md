# Analise-Espectral-Raman-com-Python
O tratamento de dados em espectroscopia é um trabalho muito custoso, devido à grande quantidade de dados amostrais. Visando diminuir, entre outros fatores, o tempo de tratamento, o laboratório está desenvolvendo um código em Python que consiga realizar o fitting de espectros. Os avanços nessa pesquisa serão divulgados neste repositório.
