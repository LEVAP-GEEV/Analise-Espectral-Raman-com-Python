# Analise-Espectral-Raman-com-Python
O tratamento de dados em espectroscopia Raman com condições termodinâmicas variadas é um trabalho muito custoso em questão de que leva-se muito tempo para chegar ao estado da arte, devido à grande quantidade de dados amostrais. Visando diminuir, entre outros fatores, o tempo de tratamento e o erro, o Laboratório de Espectroscopia Vibracional de Altas Pressões (LEVAP) do Grupo de Espectroscopia Eletrônica e Vibracional (GEEV) da Universidade Federal do Pará (UFPA) está desenvolvendo um código aberto em Python que consegue realizar o ajuste de linha de base, smooth, fitting com curvas lorentzianas de espectros e a extração de dados dessas curvas. Os demais avanços nessa pesquisa serão divulgados neste repositório.
